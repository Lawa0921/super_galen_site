文字小鎮

我是一個曾經做過很多 Side Project 的軟體工程師。 最近的夢想是將「文字小鎮」具現化——一個如同天書系統般的遊戲世界。

在這裡，創意不再只是空想，而是可以被編譯、執行、實體化的魔法。

class WorldBuilder {
  materialize(idea) {
    // Turning text into reality
    return new Reality(idea);
  }
}

Adventure Guild

我的另一個夢想，是成立一個「塞趴卷冒險公會」。就像異世界轉生番那樣，現實世界的冒險者們可以在這裡交流、組隊、接任務。

RANK S
Team Up

尋找志同道合的冒險者，一起攻克名為「現實」的副本。

RANK A
Resource Share

聚集資源，讓每個人的創意都能獲得支持與養分。

RANK B
Tavern Talk

在酒館裡分享彼此的故事，交流技術與情報。

The Warm Process

我喜歡紀錄跟寫作。把這些過程的故事分享出來，讓事情不是只有死板板的結果，而是有溫度的過程。

每一行程式碼背後，都是一個關於創造與探索的故事。
